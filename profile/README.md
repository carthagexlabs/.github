## What We're Building

### 🛡️ TanitShield — Privacy-First Credential Verification Protocol
> *Think SSL certificates, but for verified human credentials.*

TanitShield is a Zero-Knowledge credential verification protocol enabling cross-border identity and employment verification without exposing sensitive personal data. It is purpose-built for the **Tunisia–EU employment corridor**, letting employers verify qualifications and candidates prove eligibility — with mathematical privacy guarantees and no centralized data store.

- **Tech Stack:** Midnight Network · Zero-Knowledge Proofs · W3C Verifiable Credentials · DID
- **Status:** Active development · Midnight Build Club cohort
- **Docs:** coming soon
